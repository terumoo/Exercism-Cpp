# 💻 C++ Exercism Exercises

A collection of my C++ solutions to exercises from [Exercism.org](https://exercism.org/tracks/cpp).

## 📁 Structure

Each folder corresponds to an individual exercise, named after the exercise slug.

```
.
├── hello-world/
├── two-fer/
├── resistor-color/
└── ...
```

## 🛠️ Building

Most exercises can be compiled with any modern C++ compiler. Example using `g++`:

```bash
g++ -std=c++17 -o solution exercise.cpp && ./solution
```

Or using CMake if a `CMakeLists.txt` is present:

```bash
cmake -B build && cmake --build build
```

## 🚀 About Exercism

[Exercism](https://exercism.org) is a free, open-source platform for practicing coding through exercises and mentorship. The C++ track covers everything from basic syntax to more advanced concepts.
